A mobile application for a To-do List meant for Lasallians pertaining to their available subjects. This application is made to incentivize those who plan to use their app and is also part of the partial completion of the subject MOBDEVE.

Features include: 
1. Subject tasks with task prioritization and categorization
2. Local Login
3. Custom alarm and notification for a specific task
4. Sorting by title, priority, date, Filtering through searching
